# Password / PIN Verification System (Ripes)

## 📌 Description
This project simulates a Password/PIN Verification System using RISC-V assembly in the Ripes simulator.

## ⚙️ Features
- PIN comparison
- Access Granted / Denied
- 3 attempt system
- System block after failures

## 🛠️ Tools Used
- Ripes Simulator
- RISC-V (RV32I)

## ▶️ How it works
The system compares the stored PIN with entered PIN using branch instructions. If matched, access is granted; otherwise attempts decrease until system blocks.
